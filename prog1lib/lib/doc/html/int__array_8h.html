Detailed Description

An array of integers.

It stores a fixed number of integers. The prefix ia_ stands for integer array. Some operations are inherited from array.h. For example, a_length works with integer arrays and any other kind of array.

Parameters

[in]xwill be supplied to init

Example:

static int two_i_plus_x(int index, int x) {
return 2 * index + x;
}
...
Array a = ia_fn(3, two_i_plus_x, 10);
// set element 0 to init(0, 10) = 2 * 0 + 10 = 10
// set element 1 to init(1, 10) = 2 * 1 + 10 = 12
// set element 2 to init(2, 10) = 2 * 2 + 10 = 14
// now a is [10, 12, 14]

Index a is inclusive and index b is exclusive.
