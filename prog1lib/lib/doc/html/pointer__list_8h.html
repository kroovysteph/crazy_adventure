Detailed Description

A list of pointers.

Stores an arbitrary number of pointers. The prefix pl_ stands for pointer list. Some operations are inherited from list.c. For example, l_length works with pointer lists and any other kind of list.

Example 1: Creating a pointer list and printing its elements.

List a = sl_of_string("alpha, beta, gamma, delta, epsilon");
sl_println(a);
sl_free(a);

Output :

[alpha, beta, gamma, delta, epsilon]

See test functions in .c file for more examples.
