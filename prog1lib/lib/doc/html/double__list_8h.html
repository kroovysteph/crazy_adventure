Detailed Description

A list of doubles.

Stores an arbitrary number of doubles. The prefix dl_ stands for double list. Some operations are inherited from list.c. For example, l_length works with double lists and any other kind of list.

Author
Michael Rohs

10.8.2015

Create a list of n doubles, each initialized with function init.

double init(int index, double x) {}
Parameters
[in]nnumber of elements
[in]initinitialization function, will be called for each index [0, n-1)
[in]xwill be supplied to init

Example:

static double two_i_plus_x(double index, double x) {
return 2 * index + x;
}
...
List l = dl_fn(3, two_i_plus_x, 10.1);
// set element 0 to init(0, 10) = 2 * 0 + 10.1 = 10.1
// set element 1 to init(1, 10) = 2 * 1 + 10.1 = 12.1
// set element 2 to init(2, 10) = 2 * 2 + 10.1 = 14.1
// now l is [10.1, 12.1, 14.1]
