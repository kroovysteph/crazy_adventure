Detailed Description

An array of doubles.

It stores a fixed number of doubles. The prefix da_ stands for double array. Some operations are inherited from array.h. For example, a_length works with double arrays and any other kind of array.

IntDoubleToDouble  init,

)

Create an array of n ints, each initialized with function init.

double init(int index, double x) {}
Parameters
[in]nnumber of elements
[in]initinitialization function, will be called for each index [0, n-1)
[in]xwill be supplied to init

Example:

static int two_i_plus_x(int index, double x) {
return 2.0 * index + x;
}
...
Array a = ia_fn(3, two_i_plus_x, 10.0);
// set element 0 to init(0, 10.0) = 2.0 * 0 + 10.0 = 10.0
// set element 1 to init(1, 10.0) = 2.0 * 1 + 10.0 = 12.0
// set element 2 to init(2, 10.0) = 2.0 * 2 + 10.0 = 14.0
// now a is [10, 12, 14]

Index a is inclusive and index b is exclusive.
