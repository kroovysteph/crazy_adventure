Detailed Description

A list of integers.

Stores an arbitrary number of integers. The prefix il_ stands for integer list. Some operations are inherited from list.c. For example, l_length works with integer lists and any other kind of list.

Michael Rohs

Create a list of n ints, each initialized with function init.

int init(int index, int x) {}
Parameters
[in]nnumber of elements
[in]initinitialization function, will be called for each index [0, n-1)
[in]xwill be supplied to init

Example:

static int two_i_plus_x(int index, int x) {
return 2 * index + x;
}
...
List l = il_fn(3, two_i_plus_x, 10);
// set element 0 to init(0, 10) = 2 * 0 + 10 = 10
// set element 1 to init(1, 10) = 2 * 1 + 10 = 12
// set element 2 to init(2, 10) = 2 * 2 + 10 = 14
// now l is [10, 12, 14]
